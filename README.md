# Quantum-orbital-visualizer
Visualizes hydrogen atomic orbitals by: 1. Computing the wave function ψ(r, θ, φ) using quantum numbers (n, l, m) 2. Monte Carlo sampling coordinates weighted by |ψ|² (probability density) 3. Rendering the 3D point cloud, colour-coded by probability Edit N, L, M at the bottom to explore different orbitals.
